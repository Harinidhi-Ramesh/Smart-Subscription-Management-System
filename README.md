# Smart-Subscription-Management-System

A Java-based Subscription Management System that helps users track, analyze, and manage their subscriptions efficiently. The system provides personalized recommendations, budget analysis, renewal reminders, and cancellation suggestions based on usage patterns.

## Features

### User Authentication
- User Signup and Login
- Duplicate account validation
- Secure user-specific subscription management

### Subscription Management
- Add and manage subscriptions
- Support for multiple categories:
  - Entertainment
  - Music
  - Shopping
  - Academics
- Track subscription cost, usage, and validity

### Smart Analysis
- Renewal reminders for subscriptions nearing expiry
- Budget monitoring and expenditure analysis
- Cancellation suggestions for underutilized subscriptions
- Total subscription cost calculation

### Recommendation Engine
- Personalized recommendations based on:
  - User budget
  - Subscription category
  - Plan type (Individual / Family)
- Displays top affordable plans
- Separate free and paid recommendations

## Sample Workflow

1. User signs up and logs in.
2. Adds existing subscriptions with details such as cost, usage, and duration.
3. Views subscriptions by category or overall.
4. Receives renewal reminders and budget analysis.
5. Gets personalized subscription recommendations.
